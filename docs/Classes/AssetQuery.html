  • select(fields:)

    Static method for creating a new AssetQuery with a select operation: an operation in which only the fields specified in the fieldNames property will be returned in the JSON response. This variation for initializing guarantees correct query contruction by utilizing the Asset.Fields CodingKeys required by ResourceQueryable. The sys dictionary is always requested by the SDK. Example usage:

    let query = AssetQuery.select(fieldsNamed: [.file])
client.fetchMappedEntries(with: query).then { catsResponse in
    let cats = catsResponse.items
    // Do stuff with cats.
}

    See: https://www.contentful.com/developers/docs/references/content-delivery-api/#/reference/search-parameters/select-operator
